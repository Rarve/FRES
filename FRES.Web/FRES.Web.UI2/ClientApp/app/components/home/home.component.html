Welcome to Foreclosure Real Estate Search Engine


This website gather foreclosure properties from various banks in Thailand.

  • Government Housing Bank (GHB)
  • Government Savings Bank (GSB)
  • Krung Thai Bank (KTB)
  • Siam Commercial Bank (SCB)
  • Thanachart Bank

Technology Used in the website

  • .Net Core
  • AngularJS
  • Bootstrap
  • Azure Cosmos DB
  • Azure CDN

How do I gather all the properties

I basically download content in HTML for mat from each websites. HTML is then transformed into json format. The challenges are that each website has their own navigation system and HTML structure. After that, the locations (lat, lon) is retrieved by selenium from landmap

  • .Net Framework 4.6
  • SQL CE + Entity Framework
  • HTML Agility Pack
  • Selenium
  • Lot of regular expressions
