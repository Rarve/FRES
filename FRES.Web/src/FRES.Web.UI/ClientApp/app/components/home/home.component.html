Foreclosure Real Estate Search Engine (Work in Prgress)

Created by Rawitas Krungkaew - https://www.linkedin.com/in/rawitas-krungkaew/


This website gathers foreclosure properties from various banks in Thailand. I build it for my own usage (no UI of course) when I was searching for a cheap house in Thailand


Data Source

  • Government Housing Bank (GHB)
  • Government Savings Bank (GSB)
  • Krung Thai Bank (KTB)
  • Siam Commercial Bank (SCB)
  • Thanachart Bank

Website

  • .Net Core
  • AngularJS
  • Bootstrap
  • Azure Cosmos DB
  • Azure CDN

Backend and ETL Process

I basically download content in HTML for mat from each websites. HTML is then transformed into json format. The upload them to Cosmo DB. After that, the locations (lat, lon) is retrieved by selenium from landmap website

  • .Net Framework 4.6
  • SQL CE + Entity Framework
  • HTML Agility Pack
  • Selenium
  • Lot of regular expressions
